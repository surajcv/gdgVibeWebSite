# GDG Vibe Catering Website

A React website built with Vite for marketing catering products and services.

## Features

- Responsive landing page
- Services section for event catering, corporate packages, and delivery
- Featured menu items and contact booking form
- Clean, modern layout ready for Vercel deployment

## Local development

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open the URL shown in your terminal to preview the site.

## Build for production

```bash
npm run build
```

## Deploy to Vercel

1. Install the Vercel CLI or connect the repository through Vercel.
2. Run `vercel` in the project root or use the Vercel dashboard.
3. Set the framework preset to `Vite` if prompted.

The default build command is `npm run build` and the output directory is `dist`.
